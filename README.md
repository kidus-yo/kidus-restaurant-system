# 🍽️ Restaurant System

A console-based restaurant management system built with Python. This project simulates a simple restaurant ordering system where users can browse the menu, place orders, manage their cart, and perform basic restaurant operations through a menu-driven interface.

---

## 📌 Features

* 🍔 Display the restaurant menu
* 🛒 Add food items to the cart
* 👀 View items currently in the cart
* ❌ Remove items from the cart
* 🧹 Clear the entire cart
* 📋 Menu-driven console interface
* ✅ Input validation for invalid menu selections and unavailable food items

---

## 🛠️ Technologies Used

* Python 3
* Functions
* Dictionaries
* Lists
* Loops (`while`, `for`)
* Conditional Statements
* User Input Handling
